Tutorial paso a paso: Comandos principales del Símbolo del Sistema (CMD)

Guía práctica para crear archivos, navegar carpetas y usar comandos comunes — con ejemplos y un quiz al final.
Contenido rápido
  • Cómo abrir CMD
  • Comandos de navegación: dir, cd
  • Crear y borrar archivos/carpetas: type nul, echo, del, mkdir, rmdir
  • Copiar y mover: copy, xcopy, robocopy, move
  • Información de red y sistema: ipconfig, ping, tasklist, systeminfo
  • Quiz (7 preguntas)

1. Abrir el Símbolo del Sistema

  1. Presiona Windows en tu teclado o abre el menú Inicio.
  2. Escribe cmd o Símbolo del sistema.
  3. Haz clic derecho → Ejecutar como administrador si necesitas privilegios elevados (para tareas como chkdsk, format, etc.).
Consejo: también puedes usar Win + R, escribir cmd y presionar Enter.

2. Navegación básica

Estos comandos te ayudan a moverte por el sistema de archivos.

dir — listar archivos y carpetas

dir
Muestra todos los archivos y subcarpetas del directorio actual. Usa dir /a para ver archivos ocultos y del sistema.

cd — cambiar de directorio

cd \ruta\a\carpeta
cd ..      (subir una carpeta)
cd /d D:\otra\carpeta    (cambiar de unidad y carpeta)
Si solo escribes la letra de la unidad (D:) cambiarás a esa unidad sin cambiar la carpeta actual en ella.

3. Crear y editar archivos desde la consola

type nul > archivo.txt — crear archivo vacío

type nul > archivo.txt
Crea archivo.txt vacío en el directorio actual.

echo — crear archivo con contenido

echo Hola mundo > saludo.txt
Crea saludo.txt y escribe "Hola mundo". Usa >> para agregar líneas sin sobrescribir.

copy con — escribir desde la consola

copy con nota.txt
(escribe texto aquí)
Ctrl+Z + Enter    (guardar)
Abre un modo para teclear varias líneas; guarda con Ctrl+Z.

notepad archivo.txt — abrir en bloc de notas

notepad archivo.txt
Si el archivo no existe, Notepad te preguntará para crearlo.

4. Crear y eliminar carpetas y archivos

mkdir (o md)

mkdir nueva_carpeta
md otra_carpeta

rmdir (o rd) y del

rmdir nombre_carpeta           (solo si está vacía)
rmdir /s nombre_carpeta        (eliminar carpeta y todo su contenido)
del archivo.txt                 (eliminar archivo)
del /f /q archivo.txt           (/f fuerza, /q silencio)
Siempre confirma que no estás borrando archivos importantes. rmdir /s pregunta confirmación a menos que añadas /q.

5. Copiar y mover archivos

copy

copy archivo.txt D:\backup\archivo.txt
Copia archivos simples. Para carpetas y copias más robustas usa xcopy o robocopy.

xcopy — copiar directorios

xcopy carpeta_origen carpeta_destino /s /e /i
/s copia subdirectorios (excepto vacíos), /e incluye vacíos, /i asume destino como carpeta.

robocopy — copia avanzada

robocopy C:\origen D:\destino /MIR /Z /R:3 /W:5
/MIR sincroniza ( espejo ), /Z modo reiniciable, /R retry attempts, /W wait sec.

move

move archivo.txt D:\nueva_ubicacion\

6. Visualizar contenido y buscar texto

type archivo.txt

type archivo.txt

find — buscar texto dentro de archivos

find "palabra" archivo.txt
find /I "texto" *.log     (buscar sin distinguir mayúsculas)

findstr — búsquedas más avanzadas con expresiones

findstr /s /i /n "error" *.log

7. Comandos útiles de sistema y red

ipconfig          (ver IPs y configuración de red)
ipconfig /all      (detalles completos)
ping 8.8.8.8       (comprobar conectividad)
tracert ejemplo.com   (rastrear ruta)
netstat -ano       (puertos y conexiones)
tasklist           (procesos activos)
taskkill /PID 1234 /F    (matar proceso por PID)
systeminfo         (información del sistema)
Usa tasklist | findstr nombre para filtrar procesos por nombre.

8. Permisos y atributos

attrib +r archivo.txt      (solo lectura)
attrib -r archivo.txt      (quitar solo lectura)
icacls carpeta /grant Usuario:F   (gestionar permisos NTFS)
Modificar permisos con icacls puede requerir privilegios de administrador.

9. Comandos para mantenimiento

chkdsk C: /f
sfc /scannow      (Comprobador de archivos del sistema)
dism /online /cleanup-image /restorehealth
Ejecuta estos comandos con privilegios de administrador.

10. Consejos prácticos y atajos

  • Tab autocompleta nombres de archivos y carpetas.
  • F7 muestra historial de comandos (interfaz gráfica minimal).
  • doskey /history muestra historial en texto.
  • Puedes arrastrar una carpeta desde el Explorador al CMD para pegar su ruta.
  • Redirecciona salidas a archivos: comando > salida.txt o comando 2>&1 > log.txt.

Quiz: 7 preguntas

Prueba tus conocimientos. Haz clic en "Mostrar respuesta" para ver la solución.

1. ¿Qué comando usas para listar archivos y subcarpetas del directorio actual?
Respuesta: dir
2. ¿Cómo creas un archivo vacío llamado notas.txt desde la consola?
Respuesta: type nul > notas.txt (también copy con notas.txt y luego Ctrl+Z para escribir contenido).
3. ¿Qué comando sirve para cambiar de unidad y carpeta al mismo tiempo?
Respuesta: cd /d D:\ruta\carpeta
4. ¿Qué comando elimina una carpeta y todo su contenido sin pedir confirmación?
Respuesta: rmdir /s /q nombre_carpeta
5. Si quieres ver qué procesos están corriendo en el sistema, ¿qué comando usas?
Respuesta: tasklist
6. ¿Cómo comprobarías la conectividad hacia google DNS (8.8.8.8)?
Respuesta: ping 8.8.8.8
7. ¿Qué comando muestra la configuración IP (direcciones IPv4, gateways, DNS)?
Respuesta: ipconfig /all
